# Shop Order Control

## Overview

This is a application in which sellers have the capacity to manage their products with basic data.

## Features

- Register user
- Products CRUD
- Authentication
- Purchase of products
- Products Report

## Endpoints

- **Products:** /product/
- **User:** /user/
- **User Type:** /user-type/
- **Order:** /order/
- **Order Detail:** /order-type/
- **Register:** /auth/register
- **Login**: /auth/login
